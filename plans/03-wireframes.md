## 9. 인터랙션 정리

| 인터랙션 | 동작 |
|---------|------|
| 테이블 행 hover | 해당 마커 하이라이트 (지도) |
| 마커 클릭 | 팝업 표시 + 테이블 행 하이라이트 |
| 팝업 "상세보기" 클릭 | 품종 상세 페이지 이동 |
| 지도 클릭 (업로드) | GPS 좌표 자동 입력 |
| 필터 변경 | 테이블 + 지도 동시 업데이트 |
| 검색 입력 | 실시간 자동완성 제안 |
| 파일 드래그앤드롭 | 업로드 영역에 파일 추가 |
| 체크박스 선택 (다운로드) | 선택 파일 수 + 총 크기 표시 |
